Trabalhar com os métodos window.alert() e window.prompt() para ler e exibir dados pode ser bem fácil e prático, porém nada bonito. A ideia é sair o mais rápido possível disso e partir para os formulários, eventos e funções. Mas como compreender esta abordagem logo de início, se não soubermos o que é um formulário, um evento ou até mesmo uma função? Então, assim que soubermos criar uma função em JS, precisamos conhecer a estrutura de um arquivo HTML, saber usar os elementos <form> , <input> e <div>, o evento onclick e o método getElementById(). Pronto! Sabendo estes detalhes, estaremos preparados para construir scripts muito mais amigáveis. Um próximo passo seria o "embelezamento" da nossa UI, porém, fica como lição de casa estudar as folhas de estilo, ou CSS.

O formulário: 
<form>
	<input id='texto1' type='text' placeholder='texto inicial'>
	<input id='botao1' type='button' value='ok'>
	<div id='saida'>Mostra o resultado aqui!</div>
</form>
O script: 
function copiar(){
    var texto = document.getElementById('texto1').value;
    if (texto != ''){
        document.getElementById('saida').innerHTML = 'Você digitou: <b>'+texto+'</b>';
    } else {
        document.getElementById('saida').innerHTML = 'Digite algo!';
    }
}
